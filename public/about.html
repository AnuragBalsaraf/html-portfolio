Projects

Project 1: Task Manager Application

Description: A full-featured task manager application that allows users to create, update, delete, and manage tasks. Users can categorize tasks, set deadlines, and prioritize them.

Technologies Used: MERN stack (MongoDB, Express.js, React.js, Node.js).

Role: As the lead developer, I designed and implemented the entire application. I focused on creating a responsive and intuitive user interface with React and managing the backend services with Node.js and Express. The data was stored in MongoDB, which allowed for flexible and scalable data management.

Features: User authentication, task categorization, priority setting, deadline reminders, and a dashboard to visualize task status and progress.

Project 2: E-Commerce Platform

Description: An e-commerce website where users can browse products, add them to the cart, and make purchases. The platform also includes an admin panel for managing products, categories, and orders.

Technologies Used: Python (Django), HTML, CSS, JavaScript, PostgreSQL.

Role: I was responsible for developing the backend using Django, setting up the database schema with PostgreSQL, and creating APIs for the frontend to consume. I also integrated third-party payment gateways and implemented user authentication and authorization.

Features: Product listing, shopping cart, order management, payment integration, user accounts, and an admin dashboard for inventory and order management.

Project 3: Social Media Platform

Description: A social media platform where users can create profiles, post updates, follow other users, and interact through likes and comments.

Technologies Used: MERN stack (MongoDB, Express.js, React.js, Node.js).

Role: As a full stack developer, I worked on both the frontend and backend. I designed the user interface with React and handled state management using Redux. On the backend, I set up RESTful APIs with Express and Node.js and managed data with MongoDB.

Features: User registration and login, profile creation and editing, posting updates, following/unfollowing users, like and comment functionalities, and real-time notifications.
